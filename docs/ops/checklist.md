# 开发检查清单

## P0 验收

- [ ] 摄像头预览可见。
- [ ] 每一轮对话都会采集一张压缩截图。
- [ ] 麦克风 VAD 在短暂停顿后自动提交语音。
- [ ] ASR 返回 `userText`。
- [ ] 前端携带 `userText`、`snapshot` 和 `sessionId` 调用 `POST /api/chat`。
- [ ] AI 响应包含 `replyText`。
- [ ] AI 响应包含视觉观察，或包含视觉失败原因。
- [ ] UI 在 AI 回复旁展示视觉依据。
- [ ] 截图或 turn JSON 上传到七牛云。
- [ ] session JSON 可以表达一段完整演示对话。

## 降级检查

- [ ] 摄像头权限被拒绝时，仍允许纯语音对话。
- [ ] 截图失败不阻塞文本回复。
- [ ] 视觉 API 失败时展示清晰的降级状态。
- [ ] TTS 失败不阻塞下一轮对话。
- [ ] 音乐生成失败时返回重试入口或预生成降级素材。

## 演示检查

- [ ] AI 至少提到一个可见物体或环境线索。
- [ ] AI 在同一条回复中结合视觉线索和语音意图。
- [ ] 至少一个产物展示或使用了七牛云 CDN URL。
- [ ] README 说明成本控制。
- [ ] 没有提交真实 API Key。
